# 扫雷微信小程序部署指南

## 📋 前置条件

- 安装微信开发者工具
- 注册微信小程序账号（可选，可使用测试号）

## 🚀 部署步骤

### 1. 打开项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录 `C:\111\505\1`
4. 填写 AppID（若无，可选择「测试号」）

### 2. 开通云开发

1. 点击左侧工具栏的「云开发」
2. 点击「开通」按钮
3. 选择「按量付费」或「包年包月」
4. 等待云开发环境创建完成
5. 记录环境ID

### 3. 配置云环境

1. 打开 `miniprogram/app.js` 文件
2. 将 `your-env-id` 替换为你的云开发环境ID

```javascript
wx.cloud.init({
  env: 'your-env-id',  // 替换为实际环境ID
  traceUser: true
})
```

### 4. 创建数据库集合

1. 在云开发控制台中，点击「数据库」
2. 点击「+ 集合」
3. 创建名为 `minesweeper` 的集合

### 5. 部署云函数

1. 在左侧资源管理器中，右键点击 `cloudfunctions/minesweeper`
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

### 6. 预览测试

1. 点击「预览」按钮
2. 扫描二维码在手机上测试游戏

## 🎮 游戏玩法

### 基本操作
- **点击**：揭开格子
- **长按**：标记/取消标记地雷

### 游戏规则
1. 点击格子揭开，数字表示周围8格内的地雷数量
2. 长按格子可标记可疑地雷（🚩）
3. 揭开所有非地雷格子即可获胜
4. 点击到地雷则游戏结束

### 难度设置
- **简单**：9×9 格子，10个地雷
- **中等**：16×16 格子，40个地雷
- **困难**：16×30 格子，99个地雷

## 📊 数据存储

游戏数据存储在云数据库 `minesweeper` 集合中：

| 字段 | 类型 | 说明 |
|------|------|------|
| time | number | 游戏用时（秒） |
| difficulty | string | 难度等级 |
| won | boolean | 是否获胜 |
| createdAt | date | 创建时间 |

## 🔧 调试技巧

1. 在微信开发者工具中打开「调试器」
2. 查看 Console 面板输出日志
3. 在云开发控制台查看数据库记录

## ❓ 常见问题

**Q: 云开发环境创建失败？**
A: 检查网络连接，确保微信开发者工具已登录

**Q: 数据库读写失败？**
A: 检查集合权限设置，确保已开启读写权限

**Q: 游戏界面显示异常？**
A: 检查 `app.json` 中页面路径配置是否正确
